Formatos compatibles

JPG Recomendado

JPEG/JPG

Ideal para fotografías y imágenes con muchos colores.

  • • Excelente compresión
  • • Amplia compatibilidad
  • • Tamaño máximo: 50MB
PNG Transparencia

PNG

Perfecto para logos, iconos y imágenes con transparencia.

  • • Soporte de transparencia
  • • Sin pérdida de calidad
  • • Tamaño máximo: 50MB
WebP Moderno

WebP

Formato moderno con excelente compresión y calidad.

  • • Mejor compresión que JPG
  • • Soporte de transparencia
  • • Tamaño máximo: 50MB
GIF Animaciones

GIF

Para imágenes animadas y gráficos simples.

  • • Soporte de animación
  • • Limitado a 256 colores
  • • Tamaño máximo: 25MB
TIFF Profesional

TIFF

Formato profesional para impresión de alta calidad.

  • • Máxima calidad
  • • Archivos grandes
  • • Tamaño máximo: 100MB
BMP Básico

BMP

Formato básico sin compresión.

  • • Sin compresión
  • • Archivos muy grandes
  • • Tamaño máximo: 50MB

Consejos de optimización

Mejores prácticas

Usa la resolución adecuada

Para web: 72-96 DPI. Para impresión: 300 DPI o más.

Elige el formato correcto

JPG para fotos, PNG para gráficos con transparencia, WebP para web moderna.

Equilibra calidad y tamaño

Calidad 85-95% ofrece el mejor balance entre tamaño y calidad visual.

Procesa por lotes

Sube múltiples imágenes con configuraciones similares para ahorrar tiempo.

Tiempos de procesamiento

Imágenes pequeñas (<1MB) 5-15 segundos
Imágenes medianas (1-5MB) 15-45 segundos
Imágenes grandes (5-25MB) 45-120 segundos
Imágenes muy grandes (>25MB) 2-5 minutos

Los tiempos pueden variar según la carga del servidor y la complejidad de la imagen.

Preguntas frecuentes

Todas las imágenes se procesan de forma segura y se eliminan automáticamente de nuestros servidores después de 24 horas. Utilizamos conexiones HTTPS encriptadas y no almacenamos ni compartimos tus archivos con terceros.

El tamaño máximo por archivo es de 50MB para la mayoría de formatos (JPG, PNG, WebP, BMP) y 100MB para archivos TIFF. Para GIF animados, el límite es de 25MB. Si necesitas procesar archivos más grandes, contáctanos.

Sí, puedes subir hasta 10 imágenes simultáneamente. Todas se procesarán con la misma configuración que elijas. Para lotes más grandes, considera usar nuestra API o contáctanos para opciones empresariales.

Si el procesamiento falla, recibirás un mensaje de error específico. Los errores más comunes incluyen archivos corruptos, formatos no soportados o archivos demasiado grandes. Puedes intentar nuevamente o contactar soporte si el problema persiste.

Para obtener la mejor calidad, usa imágenes de alta resolución como fuente, elige el formato adecuado para tu uso (JPG para fotos, PNG para gráficos), y ajusta la configuración de calidad entre 85-95%. Evita redimensionar a tamaños mucho más grandes que el original.

Solución de problemas

Problemas comunes

Error: "Archivo no soportado"

El formato de tu archivo no es compatible.

Solución:
  • Verifica que sea JPG, PNG, WebP, GIF, TIFF o BMP
  • Asegúrate de que la extensión sea correcta
  • Intenta convertir el archivo en otro programa primero

Error: "Archivo demasiado grande"

Tu archivo excede el límite de tamaño permitido.

Solución:
  • Reduce el tamaño del archivo antes de subirlo
  • Usa un formato más eficiente (WebP en lugar de PNG)
  • Contacta soporte para archivos especiales

Procesamiento muy lento

El procesamiento está tardando más de lo esperado.

Solución:
  • Verifica tu conexión a internet
  • Evita procesar múltiples archivos grandes simultáneamente
  • Intenta en horarios de menor demanda

Códigos de error

ERROR_005 Servidor ocupado

Si encuentras un error no listado aquí, por favor contáctanos con el código específico.

Documentación API

Introducción

La API de Charlie te permite integrar nuestro procesamiento de imágenes directamente en tus aplicaciones. Es RESTful, usa JSON y requiere autenticación por API key.

Endpoint base

https://api.charlie-mx.com/v1/

Autenticación

Incluye tu API key en el header de autorización:

Authorization: Bearer tu_api_key_aqui

Obtener API Key

Regístrate para obtener tu clave API gratuita con 1000 procesamiento mensuales.

Ejemplo de uso

# Procesar imagen
curl -X POST \
https://api.charlie-mx.com/v1/process \
-H "Authorization: Bearer tu_api_key" \
-H "Content-Type: multipart/form-data" \
-F "file=@imagen.jpg" \
-F "quality=85" \
-F "format=webp"

Respuesta

{
"success": true,
"job_id": "abc123",
"status": "processing",
"estimated_time": 30
}

Parámetros disponibles

quality 1-100 (defecto: 85)
format jpg, png, webp
width Ancho en píxeles
height Alto en píxeles
